# Overview Template

Use sibling markdown files:

- `src/problems/<slug>/solutions/<solution-id>.md`

The UI supplies the `AI Overview` heading. The markdown file should start directly with body content, usually `###` sections.

Recommended template:

```md
### Approach
Explain how the concrete implementation works in this repo.

### Complexity
- Time: `O(...)`
- Space: `O(...)`

### Tradeoffs
Note why someone might prefer this solution, or what limitations it has.
```

Keep the output short enough to read comfortably beside the code panel.
