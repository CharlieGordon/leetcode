# Reverse String

Given an array `s` where each item is a single character, reverse the array in place.

Do not allocate another array for the answer. Modify `s` directly using constant extra space.

## Example 1

```text
Input: s = ["h","e","l","l","o"]
Output: ["o","l","l","e","h"]
```

## Example 2

```text
Input: s = ["H","a","n","n","a","h"]
Output: ["h","a","n","n","a","H"]
```

## Constraints

- `1 <= s.length <= 10^5`
- `s[i]` is a printable ASCII character.
