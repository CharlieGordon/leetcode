# Factorial

Write a function `factorial` that accepts a non-negative integer `n` and returns
the product of every positive integer from `1` through `n`.

The factorial of `0` is `1`.

## Example 1

```text
Input: factorial(3)
Output: 6
```

## Example 2

```text
Input: factorial(5)
Output: 120
```

## Example 3

```text
Input: factorial(0)
Output: 1
```

## Constraints

- `0 <= n <= 12`
- `n` is an integer.
